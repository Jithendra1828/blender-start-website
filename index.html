Module 02

HOW TO START TODAY

7 exact steps from first install to your first render. Follow in order — don't skip any step.

01

Download & Install Blender

Go to blender.org and download the latest stable version. Completely free. Install like any normal software. Never download from third-party sites.

💡 blender.org → Download → Latest Stable Release
02

Enable GPU Rendering — Critical!

Go to Edit → Preferences → System → Cycles Render Devices. Select OptiX and tick your RTX GPU. This makes rendering 8–10x faster. Do this before anything else.

💡 Edit → Preferences → System → OptiX → tick GPU → Save Preferences
03

Spend 2 Days on Navigation Only

Don't jump into modeling yet. Spend two full days just moving around the viewport. Orbit, zoom, pan. Open the default scene and practice until navigation feels natural.

💡 MMB = orbit · Scroll = zoom · Shift+MMB = pan · Num0 = camera view
04

Complete the Donut Tutorial — Non-Negotiable

Search YouTube for "Blender Guru Donut Tutorial". This 14-part free series teaches modeling, shading, lighting, and rendering in one complete project. Always have Blender open while watching — pause and copy every single step.

💡 Never just watch — always have Blender open and follow along simultaneously
05

Install BlenderKit for Free Assets

BlenderKit gives you thousands of free furniture, materials, and plants. Go to blenderkit.com and follow the install guide. This means you don't need to model every chair and sofa from scratch.

💡 Edit → Preferences → Add-ons → Install → search BlenderKit → enable it
06

Build Your First Room

Create a simple box room: 4 walls, floor, ceiling. Cut window holes with Inset + Extrude. Import BlenderKit furniture. Add a free HDRI from Polyhaven.com for sky lighting. This is your first portfolio piece.

💡 Target: complete your first room render within 30 days of starting Blender
07

Press F12 — Your First Render!

In Render Properties, set Device to GPU Compute. Set samples to 128. Press F12 and watch your RTX create photorealistic light. Then save immediately: Image → Save As → PNG from the render window.

💡 F12 = start render · Image → Save As = save result · Don't close without saving!
Module 03

THE BLENDER INTERFACE

Know every panel before you start building. Hover each zone to learn what it does.

BLENDER 4.x VIEWPORT LAYOUT
📋 TOPBAR File · Edit · Render · Window · Help menus + current Mode selector
🛠️ TOOLBAR Press T. Move, Rotate, Scale, Annotate tools
🌐 3D VIEWPORT Your main workspace. Everything you build lives here.
⚙️ N-PANEL Press N. Exact XYZ position, rotation, scale values
🎬 TIMELINE · OUTLINER · PROPERTIES Bottom: animation timeline · Top-right: scene outliner · Right panel: render, material, modifier settings
🟠

Object Mode

Default mode. Move, scale, rotate whole objects. Press Tab to switch to Edit Mode.

🔵

Edit Mode

Press Tab. Edit the actual mesh — vertices, edges, faces. This is where you build and shape geometry.

🟣

Sculpt Mode

Paint and sculpt organic shapes like clay. Great for terrain, character details, organic bumps.

🟢

Rendered View

Press Z → Rendered. See a live GPU render preview in the viewport. Heavy on your GPU but beautiful.

Module 04

MOUSE CONTROLS

Blender is built around a 3-button mouse. Every button has a distinct, critical purpose. Master these before shortcuts.

👆

Left Click

Select objects. Click empty space to deselect. This places your 3D cursor when used in empty viewport.

LMB = Select
👉

Right Click

Opens context menu with quick actions like Shade Smooth, Set Origin, and object operations.

RMB = Context Menu
🔘

Middle Mouse (Hold + Drag)

ORBIT — rotate your view around the scene. The single most used navigation action. Hold and drag in any direction.

MMB drag = Orbit
🖱️

Scroll Wheel

Zoom in and out. Scroll towards an object to zoom in, away to zoom out. Works in all views.

Scroll = Zoom
⇧

Shift + Middle Mouse

PAN the view — slide left, right, up, down without rotating. Hold Shift while dragging MMB.

Shift + MMB = Pan
⌃

Ctrl + Middle Mouse

Smooth zoom. Alternative to scroll when you need more precise zoom control on detailed scenes.

Ctrl + MMB = Smooth Zoom
⇧

Shift + Left Click

Add objects to selection. Hold Shift and click multiple objects to select them all simultaneously.

Shift + LMB = Multi-select
⎇

Alt + Left Click (Edit Mode)

Select an entire edge loop in one click. Extremely powerful for selecting walls, door frames, floor edges.

Alt + LMB = Loop Select
💡
No Middle Mouse Button on Your Laptop?

Go to Edit → Preferences → Input → enable "Emulate 3 Button Mouse". Then Alt + Left Click acts as Middle Mouse. Works perfectly on all laptop touchpads.

Module 05

KEYBOARD SHORTCUTS

Every shortcut explained with what it does AND why you need it. Start with Basics — expand from there.

G

Grab / Move

Move selected object freely. After G → press X, Y, or Z to constrain to that axis.

Most used key. You'll press this 100x per session.
S

Scale / Resize

Resize the selected object. S → X/Y/Z to scale on one axis only.

Resize walls, furniture, any object instantly.
R

Rotate

Rotate object. R → Z → 90 rotates exactly 90° on the Z axis.

Rotate furniture, align walls, spin cameras.
Ctrl+Z

Undo

Undo the last action. Unlimited undo steps available.

Your safety net. Use it constantly — no shame!
Ctrl+S

Save File

Save your entire .blend project file.

Press this every 10 minutes. Blender can crash on heavy scenes.
Shift+A

Add Object

Opens the Add menu. Add meshes, lights, cameras, empties.

Everything in your scene starts from this single menu.
Tab

Toggle Edit Mode

Switch between Object Mode and Edit Mode.

Edit Mode is where you build and reshape geometry.
X

Delete

Delete selected object or geometry with a confirm popup.

Quick cleanup — popup prevents accidental deletion.
Shift+D

Duplicate

Copy selected object. Then move with mouse. Enter to confirm.

Duplicate chairs, lights, windows in one keystroke.
Alt+D

Linked Duplicate

Duplicate but share same mesh data — change one, change all.

Perfect for repeating identical furniture across a room.
Ctrl+A

Apply Transforms

Bake Scale / Rotation / Location into the object data.

Always apply scale before UV unwrapping or rendering!
Ctrl+J

Join Objects

Merge multiple selected objects into one single object.

Combine all room parts into one clean object for exporting.
Num 0

Camera View

See exactly what your camera sees before rendering.

Essential before every F12 render — preview your composition.
Num 1

Front View

Snap to front orthographic view.

Precise side-on positioning of walls and objects.
Num 7

Top View

Bird's-eye floor plan view of your scene.

Essential for room layout — like drawing a floor plan.
Num 3

Side View

Right-side orthographic view.

Check wall heights and vertical proportions.
Num 5

Perspective / Ortho

Toggle between 3D perspective and flat orthographic view.

Ortho for precision modelling, Perspective for visual check.
Num .

Focus Selected

Center the view on whatever is selected.

Use this when you "lose" an object in the scene!
Num /

Local / Isolate View

Hide everything except the selected object. Press again to exit.

Focus on editing one object without visual clutter.
~

View Pie Menu

Quick access popup to all camera view angles at once.

Faster than memorizing all numpad combinations.
E

Extrude

Pull out new geometry from faces, edges, or vertices.

How you build walls, columns, roofs — the most used Edit Mode key.
Ctrl+R

Loop Cut

Add a new edge loop. Hover mesh → scroll to add more cuts → click to place.

Add detail exactly where needed without extra polygons elsewhere.
I

Inset Face

Create a smaller face inside a selected face. Pull inward.

Create window holes, door frames, wall insets, ceiling panels.
Ctrl+B

Bevel

Round off sharp edges. Scroll to add more bevel segments.

Makes furniture look real and physical — not sharp and plastic.
K

Knife Tool

Cut custom edges anywhere on mesh. Click to place cuts, Enter to confirm.

Create custom window shapes or diagonal architectural cuts.
F

Fill Face

Select 3+ edges or vertices and press F to fill with a new face.

Close holes and gaps in your mesh instantly.
M

Merge Vertices

Merge selected vertices together. "At Center" is most common option.

Clean up mesh — remove doubles, weld overlapping vertices.
123

Vertex / Edge / Face Mode

Switch selection type in Edit Mode. 1=Vertices, 2=Edges, 3=Faces.

Switch between these constantly while modeling any structure.
O

Proportional Editing

Move vertices with a smooth falloff radius. Scroll to change size.

Create smooth terrain, curved roofs, organic shapes effortlessly.
Alt+Click

Select Edge Loop

Click any edge to select the entire surrounding loop.

Select entire walls, floor edges, door frames in one click.
A

Select All / Deselect All

Press once to select everything. Press again to deselect everything.

Quick total select or clean slate with a single key.
B

Box Select

Drag a rectangle to select all objects or geometry inside it.

Select entire sections of a room or multiple objects at once.
C

Circle Select

Paint selection with a circular brush. Scroll to resize brush.

Select organic regions of vertices in Edit Mode smoothly.
Ctrl+I

Invert Selection

Flip the selection — selected becomes deselected and vice versa.

Select "everything except this object" technique.
Shift+Click

Add to Selection

Hold Shift and click multiple objects to build a multi-selection.

Select chair + table + lamp together before moving or grouping.
Ctrl+Click

Select Shortest Path

In Edit Mode — selects the shortest path between two vertices/edges.

Great for selecting specific edge rows or creating clean loops.
F12

Render Image 🎨

Start the final render. Your GPU kicks in and photorealism begins.

The most satisfying key in Blender. Your goal every session.
Ctrl+F12

Render Animation

Render all frames as a complete video sequence.

For architectural walkthroughs — your premium freelance offering.
F11

View Last Render

Reopen the last rendered image without re-rendering.

Review results and compare with previous version.
Space

Play Animation

Play or pause animation preview in the viewport.

Preview camera motion before committing to a full render.
Z

Shading Pie Menu

Switch between Wireframe, Solid, Material Preview, Rendered views.

Toggle render preview instantly — use constantly during work.
H

Hide Selected

Hide selected objects from the viewport temporarily.

See inside rooms while modeling by hiding outer walls.
Alt+H

Unhide All

Reveal all hidden objects in the scene at once.

Restore everything you hid while modelling interior details.
N

N-Panel Sidebar

Toggle right sidebar showing exact XYZ values and item properties.

Precise numerical positioning — type exact room measurements.
T

Toolbar (Left)

Toggle left toolbar with all tool buttons visible.

Beginners need this visible — shows all available tools clearly.
G+X

Axis Locking

After G/S/R press X, Y, or Z to constrain movement to that axis.

Move walls exactly horizontally, scale only vertically, etc.
Module 06

SAVING YOUR WORK

Blender can crash on heavy scenes. Knowing how and when to save protects months of hard work.

💾

Save Project File

Saves your entire .blend — all objects, materials, lighting, camera, and settings. Your main working file.

Ctrl + S
📁

Save As New Version

Save a new copy with a new name. Always version: room_v1.blend, room_v2.blend. Never overwrite your only file.

Ctrl + Shift + S
🖼️

Save Rendered Image

After F12, the render is NOT auto-saved. You MUST manually save from the render window before closing.

Render Window → Image → Save As
🔄

Enable Auto-Save

Blender auto-saves to a temp folder. Enable it and set interval to 2 minutes immediately after installing.

Edit → Preferences → Save & Load
📦

Pack External Files

If your scene uses external textures, pack them into the .blend file so it works on any computer.

File → External Data → Pack Resources
🗄️

Export for Other Software

Export your scene to FBX, OBJ, or GLTF format to use in Unreal Engine, Unity, or share with clients.

File → Export → FBX / OBJ / GLTF
⚠️
Critical Warning — Read This Once

1. After rendering, save the image manually — closing the render window loses it forever.
2. Save multiple versions (v1, v2, v3) — never overwrite your only file.
3. Auto-Save recovery files are in %TEMP% on Windows or /tmp/ on Linux/Mac.
