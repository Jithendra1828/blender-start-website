# 🧊 Blender Start Website

A beginner-friendly Blender learning website that provides essential shortcuts, controls, and learning resources for new users.

This project acts as a **quick reference guide for Blender beginners**, helping them understand keyboard shortcuts, navigation controls, and basic workflow.

---

# 📌 Features

✔ Beginner Blender shortcuts
✔ Interactive UI with animations
✔ Animated 3D cube hero section
✔ Scroll animations
✔ Organized shortcut syllabus
✔ Clean modular frontend architecture
